# Audio Files

Place your audio files in this directory:

- **ambience.mp3** - Background ambient music (looping)
- **click.mp3** - Click sound effect
- **hover.mp3** - Hover sound effect (for buttons/links)
- **menu-open.mp3** - Menu opening sound
- **menu-close.mp3** - Menu closing sound
- **transition.mp3** - Page transition sound

## Recommended Sources for Free Audio

- [Freesound.org](https://freesound.org/)
- [Mixkit.co](https://mixkit.co/free-sound-effects/)
- [Zapsplat.com](https://www.zapsplat.com/)
- [SoundBible.com](http://soundbible.com/)

## Tips

- Keep file sizes small (< 100KB for sound effects)
- Use MP3 format for browser compatibility
- Volume levels are pre-configured in AudioManager.tsx:
  - Ambience: 0.3 (30%)
  - Click: 0.4 (40%)
  - Hover: 0.2 (20%)
  - Menu: 0.5 (50%)
